# Scenia Motion

**Creative coding on the modern web — inspired by ActionScript 3, built for the browser.**

[Scenia Motion](https://github.com/scenia-motion) is home to **Scenia**, an experimental runtime that explores whether the feel of AS3 creative coding — display lists, sprites, `Event.ENTER_FRAME`, code-driven animation — can live again on the web. User code compiles to WebAssembly via AssemblyScript; the host renders with Canvas2D.

This is **not** a Flash emulator. We are not targeting SWF compatibility or a full AS3 standard library. The goal is a small, ergonomic runtime for motion and interactive graphics.
